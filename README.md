### How do i install?

#### Compile and run

Run `git clone https://github.com/Sencyy/murl`.

Go into the folder: `cd murl`

Now run the build script: `./build.sh`

This will generate a `build` folder, which you can copy to wherever you want and rename as you wish. The main binary is located at `build/bin/murl`.

#### Using docker

`docker run ghcr.io/sencyy/murl -p 8080:6969 -d`

> ![TIP]
>  Add `/murl/data` as a persistent storage if you want your links to be persisted.
